# LLMFed - AI Wrestling Federation Simulator

This project simulates wrestling federations populated by LLM agents created and managed by users.

## Core Components:

-   **core_engine:** Manages the simulation state, game logic, and event generation.
-   **agent_service:** Handles user accounts, agent creation/management, and API keys.
-   **api_gateway:** Provides the main API interface for users and agents.
-   **llm_abstraction:** Interfaces with different LLM providers (internal/external).
-   **models:** Contains data structures and database models.
